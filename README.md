# App Radar Installers

This repository contains the official installers and release artifacts for the App Radar suite of applications.

## Included Apps

- **Room Radar**: Smart room and space management.
- **Task Radar**: Efficient task and project tracking.
- **Fun Radar**: Entertainment and activity discovery.
- **Bargain Radar**: Deal and discount finder.

## Platforms Supported

Each application provides installers for the following platforms:
- **Windows** (.msi)
- **Android** (.apk, .aab)
- **iOS** (Coming soon)
- **MacOS** (Coming soon)

## How to Install

Navigate to the specific app directory (e.g., `RoomRadar/Windows/`) and download the latest installer for your platform.
